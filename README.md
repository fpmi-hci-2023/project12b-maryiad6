# Project

Веб-сайт блога по теме "Культура Беларуси"

# Description

Разработать веб-сайт “Культура Беларуси”. 
Тип сайта  —  блог. 
Используется язык разработки Python и фреймворк Django. 
В приложении можно создавать статьи, категории статей, теги.
